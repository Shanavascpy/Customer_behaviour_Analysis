Customer Behaviour Analysis

Project Overview
This project focuses on analyzing customer behaviour using data analysis techniques and business intelligence tools. The objective is to clean raw customer data, extract meaningful business insights, and create interactive visualizations to support decision-making.

The project workflow includes:
- Python for data cleaning and preprocessing
- SQL for business insights and querying
- Power BI for dashboard creation and visualization
Tools & Technologies Used

• Python – Data cleaning and preprocessing

• Pandas – Data manipulation

• NumPy – Numerical operations

• SQL – Data querying and business insights

• Power BI – Data visualization and dashboarding

• Excel/CSV – Data source
